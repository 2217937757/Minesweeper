# 扫雷游戏 (Minesweeper)

经典的 Windows 7 风格扫雷游戏，使用 Python 和 Tkinter 开发。

## 功能特性

### 🎮 经典玩法
- **左键点击**：揭示格子
- **右键点击**：标记/取消标记地雷（🚩）
- **数字快速开格**：点击已揭示的数字，如果周围标记的地雷数等于该数字，自动揭开其他格子

### 😊 表情反馈
- **😊**：正常状态
- **😮**：按下鼠标时
- **😵**：踩到地雷，游戏失败
- **😎**：成功获胜

### 📊 难度选择
- **初级**：9×9 网格，10个地雷
- **中级**：16×16 网格，40个地雷
- **高级**：16×30 网格，99个地雷

### ⏱️ 计时系统
- 红色 LED 风格数字显示
- 第一次点击时开始计时
- 最大显示 999 秒

### 💣 地雷计数器
- 实时显示剩余地雷数（总地雷数 - 已标记数）
- 支持负数显示（当标记数超过地雷数时）

### 🎨 经典界面
- Windows 7 风格的灰色 3D 边框
- 经典配色方案
- 不同数字使用不同颜色显示

## 操作说明

### 键盘快捷键
- **F2**：重新开始游戏

### 菜单选项
- **游戏 → 开局**：重新开始当前难度
- **游戏 → 初级/中级/高级**：切换难度
- **游戏 → 退出**：关闭游戏

### 高级技巧
1. **快速开格（Chord Click）**：
   - 当一个数字周围的标记地雷数等于该数字时
   - 左键点击该数字可以一次性揭开所有未标记的相邻格子
   - 这是高手必备的技巧！

2. **标记策略**：
   - 确定的地雷用右键标记
   - 不确定的格子先留着
   - 利用数字推理找出安全区域

## 运行要求

- Python 3.6+
- Tkinter（通常随 Python 一起安装）

## 启动游戏

```bash
python main.py
```

## 游戏规则

1. 游戏目标：找出所有地雷而不触发它们
2. 数字表示周围8个格子中的地雷数量
3. 标记所有地雷并揭开所有安全格子即可获胜
4. 第一次点击永远不会是地雷

## 技术实现

- **GUI框架**：Tkinter
- **数据结构**：二维数组存储游戏板状态
- **算法**：递归展开空白区域
- **事件处理**：鼠标点击、释放、悬停事件

## 开发说明

代码结构清晰，主要类和方法：
- `Minesweeper`：主游戏类
- `create_menu()`：创建菜单栏
- `create_widgets()`：创建游戏界面
- `place_mines()`：放置地雷（确保第一次点击安全）
- `reveal_cell()`：揭示格子（含递归展开）
- `chord_click()`：快速开格功能
- `check_win()`：检查胜利条件
